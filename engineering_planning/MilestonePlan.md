## Milestone Exit Criteria (Sign-off Conditions)

### Milestone 0: Project Setup
*   Code compiles locally without errors.
*   Next.js app is deployed to a staging environment on Vercel.

### Milestone 1: Foundation
*   Navbar and Footer render correctly across mobile and desktop screens.
*   Theme settings persist across page reloads.

### Milestone 2: Homepage
*   Home page loads and renders all nine required sections correctly.
*   Main call-to-actions are visible on mobile screens.

### Milestone 3: Projects
*   Project details template page renders correctly for both residential and farmland layouts.

### Milestone 4: Gallery
*   Lightbox expands images smoothly and closes on Esc key presses.

### Milestone 5: Contact
*   Contact page forms validate field inputs.

### Milestone 6: Backend
*   Form submissions are successfully logged to the Supabase database.
*   Email alerts trigger correctly when new leads are captured.

### Milestone 7: SEO
*   Lighthouse audits verify dynamic meta tags and schema markup.

### Milestone 8: Testing
*   Form submissions and page navigation paths are tested and verified.

### Milestone 9: Deployment
*   The live website resolves correctly on the custom domain with active SSL validation.
