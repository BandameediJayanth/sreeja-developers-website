## Quick Reference

### Colors

| Name | Hex | Usage |
|------|-----|-------|
| Obsidian Charcoal | `#0E1013` | Dark backgrounds |
| Telangana Gold | `#D4AF37` | CTAs, accents, RERA badges |
| Haritha Green | `#1E4620` | Parks, eco, available plots |
| Clear Ivory | `#FBFBFC` | Light sections, legal docs |

### Typography

| Role | Font |
|------|------|
| Display / H1–H2 | Cinzel (caps, 0.12em tracking) |
| UI / Buttons | Outfit (semi-bold, 0.05em) |
| Body / Data | Inter (1.6 line-height) |

- ❌ "Cheap plots" / discount language / urgency spam
